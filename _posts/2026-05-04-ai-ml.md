## 🍎 Apple Silicon / on-device

- **Ollama MLX backend — M5 acceleration (preview since March 31, ongoing):** Ollama 0.19's shift from the llama.cpp Metal path to Apple's MLX framework as the default Apple Silicon inference backend remains the dominant local-inference story. On M5 chips, Ollama leverages the new GPU Neural Accelerators to achieve up to 4× improvement in time-to-first-token vs M4 baselines, per Apple's own M5+MLX research. MLX's unified address space (CPU and GPU sharing memory without copies) is the core technical advantage for LLM prefill throughput. [via snippet] — [Ollama Blog](https://ollama.com/blog/mlx) · [Sebastian Gingter / gingter.org](https://gingter.org/2026/04/23/ollama-goes-mlx/), 2026-04-23

Apache 2.0 licence. With the Ollama MLX backend now default, this is the most capable open-weight coding agent confirmed to run locally on Apple Silicon as of this week.
