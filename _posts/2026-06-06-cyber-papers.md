## 🛡️ Cybersecurity

NVD published 48 CVEs in today's window (06-06), seven scored CVSS ≥ 7. Today's crop is dominated by WordPress-plugin flaws; the day's marquee item — CISA adding the actively-exploited SolarWinds Serv-U flaw to its KEV (Known Exploited Vulnerabilities) catalog — broke yesterday and is carried here as an ongoing watch.

- **CVE-2026-9290 — WP User Manager (WordPress): unauthenticated Local File Inclusion → code execution. CVSS 7.5 (high).** Versions ≤ 2.9.17 let an *unauthenticated* attacker make the server include and run arbitrary `.php` files via the profile-template scope (LFI = tricking an app into loading a server file it shouldn't, here pivoting to running attacker-controlled PHP). No login required is what makes this nasty; pair it with any file-upload path and it's full remote code execution (RCE — running your code on someone else's server). Fixed in a later release; patch now. Vector `AV:N/AC:L/PR:N/UI:N`. ([NVD](https://nvd.nist.gov/vuln/detail/CVE-2026-9290))

- **CVE-2026-7537 — MDJM Event Management (WordPress): arbitrary file upload → RCE, public PoC on GitHub. CVSS 7.2 (high).** Versions ≤ 1.7.8.3 do no file-type/extension/MIME checks in `mdjm_send_comm_email`, so an admin-level attacker can upload an executable file and run code. It needs high privileges (PR:H), which lowers urgency, but a working proof-of-concept exploit is already published — meaning the barrier to weaponising it is essentially zero once an attacker has a foothold. Vector `AV:N/AC:L/PR:H/UI:N/C:H/I:H/A:H`. ([NVD](https://nvd.nist.gov/vuln/detail/CVE-2026-7537); [PoC](https://github.com/d0n601/CVE-2026-7537))

- **CVE-2026-9851 — Booking Package (WordPress): account-takeover privilege escalation. CVSS 7.2 (high).** In versions ≤ 1.7.16 the `updateUser` branch of an AJAX endpoint checks only a *nonce* (a one-time anti-CSRF token) but no capability, and hard-codes the "is administrator" flag to 1 — so an Editor-level user can change the email and password of *any* account, including Administrators, and seize the whole site. A textbook "missing authorization check" bug. `[ongoing since 2026-06-02]` (third distinct WordPress account-takeover CVE this week). ([NVD](https://nvd.nist.gov/vuln/detail/CVE-2026-9851))

- **CVE-2026-34123 — TP-Link Tapo C520WS smart camera: authorization bypass. CVSS 7.0 (high).** A logic flaw in the camera's API lets a *restricted* account (e.g. a "hub user" meant for low-sensitivity actions only) disguise privileged calls as permitted ones via legitimate "method mapping," bypassing the allow-list. Impact: unauthorized device resets, config changes or disruption. Why it matters: consumer IoT cameras sit on home/SOHO networks and are routinely conscripted into botnets; TP-Link has firmware notes posted. ([NVD](https://nvd.nist.gov/vuln/detail/CVE-2026-34123))

- **CVE-2026-11413 — JingDong "JD Cloud Box" AX6600 router: remote stack buffer overflow, public exploit, vendor silent. CVSS 7.4.** A stack-based buffer overflow in `set_macfilter` (`/sbin/jdcweb_rpc`) on firmware 4.5.3.r4546 is remotely triggerable and the exploit has been *publicly disclosed*. VulDB notes the vendor was contacted early and "did not respond in any way" — so treat this as unpatched. A buffer overflow lets an attacker overrun a fixed-size memory buffer to corrupt adjacent memory and potentially hijack execution. `[single-source]` (VulDB-originated). ([NVD](https://nvd.nist.gov/vuln/detail/CVE-2026-11413))

- **KEV watch — SolarWinds Serv-U CVE-2026-28318 (CVSS 7.5, DoS) added to CISA's catalog 5 June; federal patch deadline 19 June.** Serv-U is widely-deployed managed file-transfer software. The flaw is *Uncontrolled Resource Consumption* (CWE-400): an unauthenticated POST with a `Content-Encoding: deflate` header makes the service burn resources and crash — a denial-of-service, not data theft, but exploited *in the wild*. Shodan counts >12,000 exposed Serv-U servers (Shadowserver ~3,100). Fix: Serv-U 15.5.4 Hotfix 1; US federal civilian agencies must patch by 19 June under BOD 22-01. Also recently added to KEV: a deserialization flaw in Mirasvit Full Page Cache Warmer (CVE-2026-45247, added 3 June). `[ongoing since 2026-06-06]` ([NVD](https://nvd.nist.gov/vuln/detail/CVE-2026-28318); [BleepingComputer, 2026-06-06](https://www.bleepingcomputer.com/news/security/cisa-hackers-now-exploit-solarwinds-serv-u-flaw-to-crash-servers/) [via snippet])
