## 🚀 New models, datasets & open weights

- **Cohere Command A+ 05-2026 — 218B MoE, Apache 2.0, vision, native citations (May 20–21):** Cohere's first fully open-weight frontier model and its first multimodal release. Architecture: decoder-only Sparse MoE Transformer, 218B total / 25B active parameters, 128 experts (8 active + 1 shared per token). Runs in W4A4 quantization on two NVIDIA H100s or a single B200 — an unusually low footprint for this capability tier. Context window: 256K tokens. Language coverage: 48 languages including all official EU languages. Standout feature: native citation generation ("grounding spans") that links every factual claim to a specific source document or database row, eliminating post-hoc attribution retrofits. License: Apache 2.0. The `command-a-plus-05-2026-w4a4` quantised variant appeared on Hugging Face on 18 May and was updated today (22 May). Model card: [HF](https://hf.co/CohereLabs/command-a-plus-05-2026-w4a4) (126B-parameter model class count reflects full BF16 weight; W4A4 runs at ~2× lower VRAM). — [*VentureBeat*, 21 May 2026](https://venturebeat.com/technology/cohere-cracks-lossless-quantization-and-native-citations-with-first-full-apache-2-0-licensed-open-model-command-a) [via snippet]; [*MarkTechPost*, 21 May 2026](https://www.marktechpost.com/2026/05/21/cohere-releases-command-a-a-218b-sparse-moe-model-for-agentic-workflows-that-runs-on-as-few-as-two-h100-gpus/) [via snippet] [vendor PR]

- **ByteDance Lance — 3B any-to-any multimodal, today's top HF trending model (released May 19):** Lance is a unified multimodal model covering image/video understanding, generation, and editing in a single framework. 3B active parameters (based on Qwen2.5-VL-3B-Instruct). Architecture: dual-stream Mixture-of-Experts on shared interleaved multimodal sequences, designed to decouple understanding and generation pathways while enabling joint context learning. Trained from scratch on a 128-A100 budget. The model card was last updated today and holds the highest HuggingFace trending score in this session (612) with demo spaces already live. License: Apache 2.0. Preprint: [arXiv:2605.18678](https://arxiv.org/html/2605.18678v1). Model card: [HF](https://hf.co/bytedance-research/Lance). — [*MarkTechPost*, 21 May 2026](https://www.marktechpost.com/2026/05/21/one-model-three-modalities-bytedance-releases-lance-for-image-and-video-understanding-generation-and-editing/) [via snippet] [preprint]

- **Sapient Inc HRM-Text-1B — hierarchical reasoning base model (May 17):** A 1.18B-parameter text-generation base model (pre-alignment, non-instruction-tuned) built on a novel `hrm_text` architecture implementing Hierarchical Reasoning Machines. Claims: the hierarchical decomposition improves reasoning without RLHF or SFT fine-tuning; the model is intended as a base for researchers to investigate structural alternatives to chain-of-thought prompting. Prefix-LM formulation. Apache 2.0. Preprint: [arXiv:2605.20613](https://arxiv.org/abs/2605.20613). Model card: [HF](https://hf.co/sapientinc/HRM-Text-1B) (72.5K downloads as of today). - **Ollama MLX backend — Apple Silicon throughput gains still compounding:** The Ollama 0.19 switch from the llama.cpp Metal backend to Apple's native MLX framework (released 30 March 2026) continues to reshape local inference on Apple Silicon. Current MLX architecture support in Ollama covers Qwen3.5, Qwen3.6, Gemma 4; Llama, Mistral, and Phi ports are underway. The Unsloth GGUF Qwen3.6 family (see Models section) remains the primary fall-back for llama.cpp users on these models. — [*Towards AI*, May 2026](https://pub.towardsai.net/apples-mlx-runs-local-llms-3x-faster-than-llama-cpp-until-your-context-hits-40k-715ec441afbb?gi=e500a0715e81) [via snippet]; [*Ollama Blog*, 2026](https://ollama.com/blog/mlx) [via snippet]
